# Server Messenger 

serviço de mensageria via websocket - modelo chat

### Informações gerais

- serviço usa a porta 3000;
- tela de chat fica no endpoint `localhost:3000/chat`


### Informações técnicas

desenvolvido com:
- node v16.x.x
- express v4
- jquery

### Install and Start

- npm i;
- npm start;
